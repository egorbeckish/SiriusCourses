Python входит в число самых популярных языков программирования. 
Его используют в веб-разработке и в работе с большими данными, широко применяют в области машинного обучения и искусственного интеллекта. 
Популярность языка также обусловлена простотой в освоении и универсальностью. 
Язык Python часто рекомендуют начинающим разработчикам в качестве первого языка программирования.

На курсе вы познакомитесь с синтаксисом языка Python, различными типами данных, условной инструкцией и циклами. 
Разберётесь, как в Python работает объявление функций, и научитесь писать рекурсивные функции. С нуля напишете свою первую программу!
